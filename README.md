#  CricketShell

CricketShell is a fun and fully functional custom command-line shell written in Java — with commands themed around **cricket terminology**! Whether you're listing files with `field`, deleting them with `duck`, or launching processes with `bowl`, this shell brings the spirit of the game into the terminal.

---

##  Features

-  `field` – List files (like `ls`)
-  `walk <dir>` – Change directory (like `cd`)
-  `innings` – Print current directory
-  `duck <file>` – Delete a file
-  `lineup VAR=VAL` – Set a shell variable
-  `jersey alias=command` – Define an alias
-  `bowl <cmd>` – Run a system command
-  `scorecard` – View command history
-  `replay <file>` – Execute a script file of commands
-  `score` – View memory usage
-  `umpire` – Show help
-  `retire` – Exit shell
